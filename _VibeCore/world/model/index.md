# Model Extension Layer — Governance Index (New World)

本目錄為 **新世界的存在結構延伸層（Model / Extension Layer）入口**。

本層的存在目的僅為：

- 明確宣告：新世界承認「存在結構層」的語意地位
- 提供世界在結構判斷時的最小治理錨點
- 與舊世界之 Model Extension Layer 在層級上對齊

---

## Scope of This Layer

本新世界的 Model Extension Layer：

- 不新增世界規則
- 不重新定義世界主鏈語意
- 不引入或複製舊世界的結構細節
- 不描述任何流程、初始化或執行行為

本層僅承擔一項責任：

> **宣告「存在結構判斷」在新世界中是被承認的世界層級。**

---

## Usage Declaration

目前新世界的存在結構語意：

- 已由世界主鏈（World Core）完整定義
- 無額外結構擴充需求
- 因此本層不展開進一步子文件

若未來需要對以下事項進行結構性描述：

- Project / Module 合法宣告條件
- 組合限制（Assembly Constraints）
- 結構一致性裁決（Consistency Rules）

則將以 **新增文件** 方式延伸本層，
並受世界 Change Protocol 管轄。

- 本層不影響 API World 或 Module World 的可用性
- 本層僅補足「世界結構層級完整性」
