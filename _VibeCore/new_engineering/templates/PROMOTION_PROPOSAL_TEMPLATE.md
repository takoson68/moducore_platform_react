# Promotion Proposal Template

用途：用於提出升級請求（Promotion Proposal）。

注意：
- 本文件僅為「請求」格式
- **不構成自動升級**
- 需依 `Index Gate` 列冊後方可被引用

## 建議檔名規則（Evolver-compatible）

`[VisualSlot]-RP-promotion-E###-<slug>.md`

說明：
- `VisualSlot`：兩位英文字母，僅用於視覺排序，不具裁決效力
- `RP`：Promotion Proposal 的治理身份
- `E###`：Proposal ID（例如 `E014`）

## Proposal

- Target: （要升級的候選文件或既有規範）
- FromStatus -> ToStatus: （例如 `candidate -> active`、`convention -> stable`〔僅請求〕）
