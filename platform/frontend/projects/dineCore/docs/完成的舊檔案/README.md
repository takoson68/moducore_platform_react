# 完成的舊檔案

更新日期：2026-03-04

## 1. 目錄用途

本目錄存放 DineCore 已完成、已歸檔的歷史文件。

這些文件通常屬於：

- 某一階段的規劃草案
- API / 資料模型早期版本
- 前端流程規則草稿
- 任務拆解與執行書

它們仍有追溯價值，
但預設不作為目前工程的第一閱讀入口。

## 3. 建議閱讀時機

只有在以下情況，才建議優先進入本目錄：

- 需要追溯 batch 流程是怎麼一步步設計出來的
- 需要理解某次重構的原始拆解方式
- 需要確認舊規劃與現行實作之間的差異

若只是要理解目前系統，
應先回到：

- `../README.md`
- `../DINECORE_CURRENT_STATE.md`
- `../DINECORE_PROGRESS_TRACKER.md`
- `../完成項目分類索引.md`
